현재 프로젝트(Next.js + shadcn/ui)에 Supabase Local을 활용한 Todo 앱을 구현하는 계획을 `.claude/plans/`에 마크다운으로 작성해줘.

## 기능 요구사항
- 이메일/비밀번호 인증 (회원가입, 로그인, 로그아웃)
- Todo CRUD (생성, 조회, 수정, 삭제, 완료 토글)
- 카테고리, 태그, 우선순위, 마감일
- 검색, 필터, 정렬
- 사이드바 네비게이션 + 반응형
- 설정 페이지 (다크모드, 비밀번호 변경)

## 팀 에이전트 구조
Leader / Backend / Frontend / QA 4명으로 작업을 분배한다.

## 계획 형식
- TDD: QA가 테스트 먼저 작성(RED) → 구현(GREEN) → 통합 테스트
- 각 Task에 담당자, 구체적 작업 내용, 성공기준(조건/결과), 커밋 메시지 포함
- 마지막에 agent-browser E2E 테스트 시나리오 포함
- 주요 파일 목록과 검증 방법 섹션 포함
